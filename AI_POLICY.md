# AI Policy for Jelly Swipe

This project has made heavy use of AI development tooling. From the point it was forked from its original, all code produced by the AI has had human review.

## AI Philosophy

Writing code is cheap. Reviewing and maintaining code is expensive. 

We will not accept "fully automated" AI contributions. If your contributions are coming from an agentic workflow that does not involve you reviewing the code before asking for our review, it will be rejected.

Admittedly, this is an imperfect system as its harder and harder to spot "AI slop" outright these days. We ask that contributors take this policy in good faith.
