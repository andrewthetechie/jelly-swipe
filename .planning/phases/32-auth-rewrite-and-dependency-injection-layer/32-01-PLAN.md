```python
class TestGetDbDep:
    def test_yields_connection_and_closes(self, db_path, monkeypatch):
        """get_db_dep yields a sqlite3.Connection that is closed after context exits."""
        gen = get_db_dep()
        conn = next(gen)
        assert isinstance(conn, sqlite3.Connection)
        # Connection should be usable
        row = conn.execute("SELECT 1").fetchone()
        assert row[0] == 1
        # Exhaust the generator (trigger cleanup)
        try:
            next(gen)
        except StopIteration:
            pass
        # After cleanup, connection should be closed
        # sqlite3 raises ProgrammingError when using closed connection
        import sqlite3 as _sqlite
        with pytest.raises(_sqlite.ProgrammingError):
            conn.execute("SELECT 1")
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HTTP request → `require_auth()` | Untrusted input (session cookie) crosses into auth validation |
| `check_rate_limit()` → rate state | Client IP from `request.client.host` is trust-boundary-dependent (behind proxy: X-Forwarded-For) |
| `get_provider()` → lazy import | Module-level global `_provider_singleton` accessed cross-module |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-32-01 | Elevation of Privilege | `require_auth()` | mitigate | Raises HTTPException(401) for any missing/invalid session — no path to bypass |
| T-32-02 | Spoofing | `check_rate_limit()` IP source | accept | ProxyHeadersMiddleware (Phase 31) resolves X-Forwarded-For → request.client.host; already trusted at middleware layer |
| T-32-03 | Information Disclosure | `AuthUser.jf_token` in responses | mitigate | AuthUser is only used as DI parameter — never serialized to response; routes extract fields for upstream calls only |
| T-32-04 | Denial of Service | Rate limit bypass via path manipulation | mitigate | `_infer_endpoint_key()` uses substring match; all known rate-limited paths verified against `_RATE_LIMITS` dict |
</threat_model>
