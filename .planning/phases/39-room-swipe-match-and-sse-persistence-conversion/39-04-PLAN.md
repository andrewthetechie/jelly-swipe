| SSE generator -> database runtime | Stream polls must not hold shared mutable DB state across long-lived requests |

| T-39-11 | I | stream snapshot payloads | mitigate | Read only committed persisted room state through repository queries; never emit pre-commit mutation objects from service code. |
| T-39-12 | T | SSE response framing | mitigate | Keep `EventSourceResponse` and existing heartbeat/comment framing instead of hand-rolled SSE serialization. |
