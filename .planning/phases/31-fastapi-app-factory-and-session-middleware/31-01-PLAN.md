```python
class XSSSafeJSONResponse(JSONResponse):
    """JSON response that escapes HTML-sensitive characters for XSS defense.
    
    Per OWASP recommendation, < > & are encoded as \\u003c \\u003e \\u0026
    in JSON output so that raw HTTP bodies cannot contain executable HTML tags.
    """
    def render(self, content: typing.Any) -> bytes:
        result = super().render(content)
        return (result
                .replace(b"<", b"\\u003c")
                .replace(b">", b"\\u003e")
                .replace(b"&", b"\\u0026"))
```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Client → FastAPI app | Untrusted HTTP input (headers, query params, JSON bodies, path params) |

| FastAPI app → SQLite | SQL queries with parameterized inputs (SQLi defense via parameterization) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-31-01 | S (Spoofing) | SessionMiddleware | mitigate | Session cookies signed with FLASK_SECRET; same_site=lax; https_only configurable |
| T-31-02 | T (Tampering) | XSSSafeJSONResponse | mitigate | All JSON output escapes < > & — prevents XSS via JSON responses |
| T-31-03 | T (Tampering) | ProxyHeadersMiddleware | mitigate | Trust X-Forwarded headers only from trusted reverse proxy (network-level trust) |
| T-31-04 | I (Info Disclosure) | RequestIdMiddleware | mitigate | Request IDs use random hex (secrets.token_hex) — not predictable/guessable |
| T-31-05 | I (Info Disclosure) | Error responses | mitigate | 5xx errors return generic "Internal server error" message, not stack traces |
| T-31-06 | D (DoS) | Rate limiter | mitigate | Existing rate limiter preserved with same limits; uses request.client.host |
