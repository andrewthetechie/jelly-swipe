# Plan 02 — Sync smoke, `py_compile`, deprecate pip-first narrative

<objective>
Prove the new lock resolves a runnable environment on **Python 3.13** and that existing modules still **byte-compile**. Mark **`requirements.txt`** as **non-canonical for humans** (Docker may still consume it until Phase 12 — see `10-CONTEXT.md` D-06).
</objective>

<threat_model>
| ID | Threat | Severity | Mitigation |
|----|--------|----------|------------|
| T-01 | Running app without secrets triggers misleading failures | low | Use **`python -m py_compile`** only (no Flask server start); do not log env values |
</threat_model>

<must_haves>
- `uv sync` succeeds on 3.13 after Plan 01.
- `py_compile` passes on `app.py` and `media_provider/*.py` using the uv environment.
- `requirements.txt` carries a clear “use uv” pointer for maintainers (single comment line at file top is enough).
</must_haves>
