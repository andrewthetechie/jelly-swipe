  <name>Task 1: Add pytest configuration and dependencies to pyproject.toml</name>
  <files>pyproject.toml</files>
  <action>
    Add the following to pyproject.toml:

    1. Add [project.optional-dependencies] section after the existing dependencies array:
       ```toml
       [project.optional-dependencies]
       dev = [
           "pytest>=9.0.0",
           "pytest-cov>=6.0.0",
           "pytest-mock>=3.14.0",
           "responses>=0.25.0",
           "pytest-timeout>=2.3.0",
       ]
       ```

    2. Add [tool.pytest.ini_options] section at the end of the file (per D-03):
       ```toml
       [tool.pytest.ini_options]
       testpaths = ["tests"]
       python_files = ["test_*.py"]
       addopts = "-v --tb=short"
       ```
       This configures pytest to discover tests from tests/ directory, look for test_*.py files, and provide verbose output with clean tracebacks.

    Note: Coverage configuration (COV-01) is deferred to Phase 17 per D-03 rationale.
  </action>
  <verify>grep -A 10 "\[tool.pytest.ini_options\]" pyproject.toml shows testpaths, python_files, addopts</verify>
  <done>pyproject.toml contains pytest configuration and dev dependencies</done>
</task>
