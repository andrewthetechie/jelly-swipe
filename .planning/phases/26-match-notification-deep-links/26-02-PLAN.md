    ```python
        info = get_provider().server_info()
        return jsonify({
            'userId': g.user_id,
            'displayName': g.user_id,  # Jellyfin user_id is the display identifier
            'serverName': info.get('name', ''),
            'serverId': info.get('machineIdentifier', ''),
        })
    ```

The server_info() call returns {machineIdentifier, name} from Jellyfin's /System/Info endpoint. Note: server_info() uses the provider singleton (server credentials), not the per-user token — this is correct since server info is the same for all users.
