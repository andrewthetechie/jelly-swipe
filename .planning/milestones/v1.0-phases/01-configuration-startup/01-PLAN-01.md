# Plan 01 ‚Äî Provider env validation and Plex lazy import

<objective>
Implement `MEDIA_PROVIDER` (`plex` | `jellyfin`, default `plex`), provider-specific required env validation at import time, and lazy `plexapi` imports so Jellyfin-only env can boot the process without Plex variables present (CFG-01, CFG-02, CONTEXT D-01‚ÄìD-07, D-10).
</objective>

<threat_model>
| ID | Threat | Severity | Mitigation in this plan |
|----|--------|----------|-------------------------|
| T-01 | Secrets in error messages | medium | Reuse `Missing env vars: [...]` pattern with variable **names** only, never values |
| T-02 | Open proxy via `/proxy` | high (existing) | Keep existing path prefix check; add guard so `/proxy` returns 503 when `MEDIA_PROVIDER != plex` or Plex URL/token unset |
</threat_model>

<action>
1. After stdlib/third-party imports (but **remove** top-level `from plexapi.server import PlexServer` and `from plexapi.myplex import MyPlexAccount`).
2. Add `_normalized_media_provider()` per CONTEXT: empty/unset ‚Üí `plex`; `plex`/`jellyfin` case-insensitive; any other non-empty value ‚Üí `RuntimeError` with message that only `plex` or `jellyfin` are allowed (include raw value in message for debugging).
3. Set module-level `MEDIA_PROVIDER = _normalized_media_provider()`.
4. Replace the fixed `required` list with: always require non-empty `TMDB_API_KEY` and `FLASK_SECRET` via `os.getenv`. If `MEDIA_PROVIDER == "plex"`, require `PLEX_URL` and `PLEX_TOKEN`. If `MEDIA_PROVIDER == "jellyfin"`, require non-empty `JELLYFIN_URL` (strip trailing slashes like existing `PLEX_URL` handling) and require either non-empty `JELLYFIN_API_KEY` OR both non-empty `JELLYFIN_USERNAME` and `JELLYFIN_PASSWORD`. On failure, `raise RuntimeError(f"Missing env vars: {missing}")` with `missing` as a list of string names; for the credential OR-case use the single string `JELLYFIN_API_KEY or (JELLYFIN_USERNAME and JELLYFIN_PASSWORD)` as one missing entry when neither bundle is satisfied.
5. Run this validation **before** `app = Flask(__name__)` so missing `FLASK_SECRET` still uses the same `RuntimeError` path (change line 9 to read secret only after validation passes).
6. Keep `PLEX_URL = os.getenv('PLEX_URL', '').rstrip('/')` and `ADMIN_TOKEN = os.getenv('PLEX_TOKEN')` after validation (values may be empty in jellyfin mode).
7. Add optional `JELLYFIN_URL = os.getenv('JELLYFIN_URL', '').rstrip('/')` for later phases (validated when jellyfin).
8. In `get_plex()`, if `MEDIA_PROVIDER != "plex"`, raise `RuntimeError` with a clear message that Plex library access is unavailable in jellyfin mode until later phases. Otherwise `from plexapi.server import PlexServer` inside the function then instantiate/cache as today.
9. In `add_to_watchlist()`, if `MEDIA_PROVIDER != "plex"`, return JSON error 400. Else `from plexapi.myplex import MyPlexAccount` inside the function before use.
10. In `proxy()`, if `MEDIA_PROVIDER != "plex"` or not `PLEX_URL` or not `ADMIN_TOKEN`, abort with HTTP 503.
</action>
