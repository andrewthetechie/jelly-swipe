# Plan 23-02: Migrate Existing Code to Centralized Helper

**Created:** 2026-04-26
**Status:** Planning
**Phase:** 23 — HTTP Client Centralization
**Type:** Implementation

## Overview

### Task 2: Replace TMDB calls in get_trailer()

**File:** `jellyswipe/__init__.py`

**Current Code (lines 186-191):**
```python
search_url = f"https://api.themoviedb.org/3/search/movie?api_key={TMDB_API_KEY}&query={item.title}&year={item.year}"
r = requests.get(search_url).json()
if r.get('results'):
    tmdb_id = r['results'][0]['id']
    v_url = f"https://api.themoviedb.org/3/movie/{tmdb_id}/videos?api_key={TMDB_API_KEY}"
    v_res = requests.get(v_url).json()
```

**Replacement Code:**
```python
from jellyswipe.http_client import make_http_request

# Search for movie
search_url = f"https://api.themoviedb.org/3/search/movie?api_key={TMDB_API_KEY}&query={item.title}&year={item.year}"
search_response = make_http_request(
    method='GET',
    url=search_url,
    timeout=(5, 15)  # Shorter timeout for TMDB search
)
r = search_response.json()

if r.get('results'):
    tmdb_id = r['results'][0]['id']
    v_url = f"https://api.themoviedb.org/3/movie/{tmdb_id}/videos?api_key={TMDB_API_KEY}"
    videos_response = make_http_request(
        method='GET',
        url=v_url,
        timeout=(5, 15)  # Shorter timeout for TMDB videos
    )
    v_res = videos_response.json()
```

**Changes:**
- Import `make_http_request` from `jellyswipe.http_client`
- Replace `requests.get(search_url)` with `make_http_request(method='GET', url=search_url, timeout=(5, 15))`
- Replace `.json()` call on helper result
- Add explicit timeout: `(5, 15)` for TMDB API calls (shorter than default)
- Keep existing error handling (RuntimeError and Exception blocks)

**Verification:**
- Direct `requests.get()` calls removed
- `make_http_request()` calls added
- Timeout parameters explicitly set
- Response handling unchanged
- Error handling preserved

### Task 4: Replace Jellyfin calls in jellyfin_library.py
